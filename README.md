# RspecBoolean

Welcome to rspec_boolean, with this gem you can use "be_boolean" expectation to your tests


## Installation

Add this line to your application's Gemfile in the group :development, :test:

```ruby
gem 'rspec_boolean'
```

And then execute:

    $ bundle

## Usage


```ruby
expect(false).to be_boolean
```
